# Replace the following Things:
- APIKEY_HERE: Put your API Key here
- TOKEN_HERE: Put your Bot Token here
- APIURL_HERE: Put the API Url here, example api.{URL}
- ROLEIDS_HERE: Insert role ids, that are able to scan, for multiple do a , or ; between
- IMAGE_HERE: Insert link of an Image there, the discord embed Image will be displayed there
